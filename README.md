# CCNA Labs
This repository contains my collection of CCNA Practice labs and projects, demonstrating networking fundamentals, routing, switching, and basic network security concepts.

---

## Labs
- Guided practice exercises designed to reinforce CCNA topics.
- Each lab contains:

  - README.md
  - screenshots
---

## Projects
- Independent projects applying multiple CCNA concepts together.
- These are larger scenarios meant to demonstrate problem-solving and network design skills.
