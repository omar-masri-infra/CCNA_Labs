# Lab 01: Connecting Devices
## Objective:
The objective of this lab is to connect multiple devices using the appropriate cable type based on devices types and distance between them.

## Notes:
Devices used: Laptops, switches, routers, and a server.

Cable selection:
- Straight-through copper UTP – Used to connect laptops and server to switches (different Tx/Rx pins).
- Crossover copper UTP – Used to connect switches to switches and routers to routers (same Tx/Rx pins).
- Fiber optic cable – Used to connect routers across different LANs due to long distances.

Distance guidelines:
- Copper UTP: up to 100 meters
- Multimode fiber: 100 to 500 meters
- Single-mode fiber: distances of kilometers
